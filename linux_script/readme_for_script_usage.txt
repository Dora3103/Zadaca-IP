This is a bash script written to execute programs written for this robot.
Usage: just write ./robot.sh path/to/your/file/file_name and robot sould execute the code
Make sure that script has execute permissions

Dependancies: 
	Python 3.9+
	Bash/any other POSIX compliant shell
	dz1new.py and vepar.py(in the same folder as the script)
